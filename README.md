 AI Voice Lead-Gen & Automation Pipeline
An intelligent automation system that filters high-value leads and initiates immediate AI-powered voice engagement. This project streamlines the sales funnel by connecting a CRM to a conversational AI agent.

 Workflow Logic
Lead Qualification (Airtable): The system monitors new entries. If a lead’s budget exceeds $12,000, they are automatically flagged for immediate contact.

AI Voice Call (Vapi AI): Once qualified, the system triggers an outbound call using a conversational AI assistant to introduce the company and discuss the project.

Process Handling (Make.com Tools): Includes a strategic delay to allow for call completion and data processing.

CRM Update (Airtable): Automatically logs the contact timestamp and updates the record status.

Follow-up (Gmail): Sends a professional HTML follow-up email to the lead immediately after the interaction.

Team Alert (Slack): Notifies the internal team via Slack with parsed details about the lead for further action.
